🏆 This is a multi-league puzzle!
Welcome to the Bronze League! The training wheels are off.
The board has expanded to a massive 9x9 grid, and you are now facing up to 3 opponents simultaneously in a Free-For-All!

  Goal

Be the first player to form a line of 5 of your marbles either horizontally, vertically, or diagonally.
This puzzle is based on the official board game Pentago.

  Rules

On your turn, you must perform exactly two actions:

  1. Place a marble on any empty space on the grid.
  2. Rotate a 3x3 block 90 degrees either left (counter-clockwise) or right (clockwise).

Blocks are numbered 0 to 8, and coordinates x, y from 0 to 8 as shown below:

Draws: If multiple players form a line of 5 simultaneously, it is a draw between those players. If the grid is completely filled with no lines of 5, the game ends in a global draw for all active players.

  Game Input/Output

Input for one game turn

Line 1: size, the size of the board (9).

Next size lines: strings representing the grid state. A . means empty. A number 0-3 represents a player's marble.

Output for one game turn
A single line containing 4 space-separated values: x y block dir
x, y: Coordinates to place your marble.
block: The ID of the block you wish to rotate (0 to 8).
dir: The direction to rotate (L or R).

Example: 2 1 0 R (Places marble at 2,1 and rotates block 0 right)
